# 📱 手机访问使用说明

## 快速开始

### 1. 确保网络连接
- ✅ 手机和电脑连接到**同一个 WiFi 网络**
- ✅ 确认电脑 IP 地址：运行 `check-ip.bat` 或查看下方说明

### 2. 手机访问地址
```
http://192.168.1.227:3000
```

在**手机浏览器**或**微信**中输入上述地址即可访问。

## 🔧 如果点击无反应

### 问题诊断步骤

#### 第一步：测试基础点击功能
访问测试页面：
```
http://192.168.1.227:3000/click-test.html
```
点击测试卡片，看是否能跳转到抖音。

**如果测试页面能跳转** → 说明网络正常，继续下一步诊断  
**如果测试页面也不能跳转** → 检查网络和防火墙设置

#### 第二步：检查浏览器控制台（如果可能）

**Android Chrome**:
1. 在手机 Chrome 中访问 `chrome://inspect`
2. 开启 USB 调试
3. 连接电脑查看控制台日志

**iPhone Safari**:
1. 设置 → Safari → 高级 → 网页检查器（开启）
2. 连接 Mac，使用 Safari 开发菜单查看

#### 第三步：查看控制台日志

## ❌ 常见问题及解决方案

### 问题 1: 页面无法加载
**症状**: 手机显示"无法访问此网站"

**解决方案**:
1. 检查电脑防火墙是否允许 3000 端口
   - 运行：`setup-mobile.bat`（管理员权限）
2. 确认手机和电脑在同一 WiFi 网络
3. 暂时关闭防火墙测试

### 问题 2: 点击视频无反应
**症状**: 点击视频卡片后没有任何反应

**可能原因**:
1. **微信浏览器阻止跳转** - 已修复，使用 `location.replace`
2. **视频数据未加载** - 刷新页面重试
3. **抖音链接为空** - 会弹出提示"该视频暂未添加抖音链接"

**解决方案**:
1. 刷新页面（下拉刷新或点击右上角刷新按钮）
2. 检查是否能看到视频卡片
3. 尝试不同的视频

### 问题 3: 跳转失败
**症状**: 点击后显示错误页面

**可能原因**:
1. 抖音链接格式问题
2. 网络连接不稳定

**解决方案**:
1. 检查电脑网络连接
2. 尝试在其他应用中打开抖音
3. 清除浏览器缓存后重试

## 🎯 正确使用流程

1. **打开手机浏览器或微信**
2. **输入地址**: `http://192.168.1.227:3000`
3. **选择分类**: 点击顶部"SSS 儿歌"或"亲子口语"标签
4. **筛选难度**（可选）: 点击"初级/中级/高级"按钮
5. **点击视频卡片**: 会自动跳转到抖音播放
6. **观看视频**: 在抖音 APP 中观看完整视频
