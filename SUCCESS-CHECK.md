## 📱 手机打不开？清理缓存！

如果你在手机上还是看到 404 或空白页面，请**清理缓存**：

### 方法 1：强制刷新（推荐）

**在微信浏览器中**：
1. 点击右上角「···」
2. 选择「刷新」
3. 关闭页面
4. 重新打开链接

### 方法 2：清除微信缓存

1. 打开微信 → 我 → 设置
2. 通用 → 存储空间
3. 等待加载完成
4. 点击「清理」缓存（不会删除聊天记录）

### 方法 3：使用无痕模式

**在手机浏览器中**：
1. 打开浏览器的「无痕/隐私模式」
2. 访问：`https://baby-english-pied.vercel.app`
3. 如果能正常打开，说明是缓存问题

### 方法 4：添加随机参数（绕过缓存）

在链接后面加上时间戳：
```
https://baby-english-pied.vercel.app/?t=20260323
```

或
```
https://baby-english-pied.vercel.app/qrcode.html?t=12345
```
